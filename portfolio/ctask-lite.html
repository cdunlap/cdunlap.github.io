Summary

CTask is a really simple task management system that I wrote back in 2006 to manage incoming requests from testers while I worked on Firearms 2 (never released, now defunct). After Firearms 2 development halted I released the code for CTask on GitHub. It uses SQLite and PHP and is themeable via jQuery UI themeroller (though I haven't tested that in a while).
