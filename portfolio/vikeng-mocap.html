Summary

This is a tool that I wrote for recording and reviewing motion capture data as well as gesture library data for MMIST. We used it to record new gestures into the gesture recognition database as well as review virtual locomotion (walking/running in a virtual world) information.

This project is a native application written in C++ using JUCE and Visualization Library.
